 ARCHITECTURE PROTECTION RULES (LOCKED — NEVER VIOLATE)
================================================================================
  R1. Never mix floor ownership (F1 ≠ F2 ≠ F3 ≠ F4 ≠ F5)
  R2. Never mix side ownership (Side A ≠ B ≠ C)
  R3. Dashboard NEVER becomes decision layer (request_ NOT execute_)
  R4. Side C NEVER becomes intelligence layer (store facts, NOT create them)
  R5. Captain NEVER becomes execution engine (decide, NOT execute)
  R6. Floor 3 NEVER becomes interpretation layer (quality ≠ confidence)
  R7. Floor 4 NEVER becomes decision layer (interpret, NOT approve)
  R8. Replay NEVER modifies system state (read-only)
  R9. Local-first, cloud-optional (single laptop, SQLite, localhost)
  R10. Test before integrate (no untested code enters system)
